Hero.init();

Calls the 'class' Hero's constructor (init).  
The constructor initializes the Hero 'object' (hero).

class functions are called by:
	Hero.func();
	
class properties are access by:
	hero.prop;



Format guidlines:
	capitalized word:	Hero  ==> class that has coresponding object named 'hero'
	lowercase word:		utils ==> class with no other object
