About This Site

This site is built to use HTML5, CSS3, and Javascript (jQuery) on the front end. The CSS is designed to be responsive (make the pages scale well).

It was originally built in PHP and compiled to static HTML pages. It is now a SPA and utilizes Sammy.js for routing, and custom JS for view models.
